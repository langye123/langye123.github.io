进入linux后常见命令

记住在进入mysql后一定要加;在语句结束的时候
当输入错误的时候记得\c

  1. mysql -u root -p; 进入mysql -u 指定用户登录 -p 显示密码提示
  2. show databases; 展示数据库
  3. use 数据库; 使用数据库
  4. show tables; 展示数据表
  5. describe users; 展示某个表
  6. select * from users; 查看某个表里面所有信息
  7. update users set password=md5('123') where user=('admin'); 因为MySQL里面存储密码都是MD5加密
  8. insert into users (user, password) values ('test', MD5('123456')); 新增用户并且指定某一列是什么其他默认
  9. delete from users where user='test'; 删除用户

增加修改mysql权限用户给新的用户

  1. create user 'text'@'%' identified by '123456'; 创建一个名字为text密码为123456的新用户。
  2. grant all privileges on dvwa.* to 'text'@'%'; 赋予它所有权限，并且任意地点登录。
    flush privileges; 刷新权限表
