docker BeEf搭建与常用命令

发表于2025-12-16|更新于2025-12-16|工具
|字数总计:301|阅读时长:1分钟|阅读量:

必须先安装docker

详见 安装docker部署DVWA靶场

使用官方维护的 Dockerfile（最稳）

cd /opt

git clone https://github.com/beefproject/beef.git

cd beef

拉拢之后进入目录下修改config.yaml里面的默认用户名和密码。

构建 Docker 镜像

构建Build 第一次构建会稍慢，这是在拉 Ruby 和依赖，耐心等一下

docker build -t beef .

t后面表示镜像的名字

运行 Docker 容器

运行命令

3001:3000

左边：宿主机端口（你访问的）

右边：容器内部 BeEF 端口（固定 3000）

-d：后台运行

–name beef：容器名

-p 3001:3000：Web UI 端口

查看日志 docker logs beef

  1. 删除旧镜像和容器
  2. docker stop beef && docker rm beef 删除容器
  3. docker rmi beef 删除镜像
  4. docker build -t beef .一定要有.
    重新构建（确保当前目录有新 config.yaml）

记住一定要多看docker logs beef
又错了，好像是初始化的时候一个扩展没有加载，去config.yaml里面把它改成false，先保障基本功能可用，后续再说geoip的问题

访问地址 http://ip地址:3001/ui/panel

docker stop beef

docker start beef
