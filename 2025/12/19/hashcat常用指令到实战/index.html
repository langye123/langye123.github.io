什么是hashcat

Hashcat 是一款极其强大的密码恢复工具，它能高速破解数百种哈希与加密算法。无论是破解系统哈希、攻击加密文件，还是处理你关心的 JWT，它都是核心工具

常用命令

查看设备（每次实战前必做）
hashcat.exe -I

👉 确认 CPU / GPU 是否被正确识别

查看支持的算法（查模式用）

hashcat --help | findstr <关键词>

🛠️ Hashcat 常用参数速查表（两列版）

参数 说明
-m 或 --hash-type 指定哈希类型。JWT (HS256/HS384/HS512) 使用模式 16500。例：-m 16500
-a 或 --attack-mode 指定攻击模式：
• 0 = 字典攻击（Straight）
• 1 = 组合字典（Combination）
• 3 = 掩码/暴力攻击（Brute-force）
• 6/7 = 混合模式（字典+掩码）
-w 或 --workload-profile 工作负载强度：
• 1 = 低（后台运行）
• 2 = 默认
• 3 = 高（推荐 GPU 强时使用）
• 4 = 极限性能（可能卡顿）
-O 或 --optimized-kernel-enable 启用优化内核，大幅提升速度（但限制密码长度 ≤ 32 字符），强烈建议加上！
-r 或 --rules-file 应用规则文件，对字典词进行变形（如加数字、大小写切换等）。常用规则：best64.rule, dive.rule 等。
--show 显示已破解结果，从 potfile 中直接输出已破解的 secret，无需重新跑。
-o 或 --outfile 自定义输出文件，将破解结果保存到指定文件（默认是 hashcat.potfile）。
-b 或 --benchmark 运行基准测试，查看 GPU 在各算法下的破解速度，用于性能验证。
-I 或 --device-info 显示设备信息，确认 GPU 是否被正确识别（尤其多显卡环境有用）。
--potfile-disable 禁用 potfile，不记录/读取历史破解结果（用于强制重跑或测试）。
--status 实时显示进度，包括速度、剩余时间、当前尝试值等，跑长任务时超安心～

提取hash值进行ZIP压缩包破解密码

zip2hashcat这个提取hash值的工具是专门为了hashcat而生的

命令 zip2hashcat.exe 18AA1F.zip > zip.hash

要注意它是用hashcat进行破解所以它的格式要求非常严格,用bandzip能打开的不一定能提取成功。用binwalk提取一下

成功提取之后用命令记住-m 13600

hashcat.exe -m 13600 任务\zip.hash 字典\rockyou.txt -w 3 -O

MD5加密破解
