📘 参数解析

  • -d：后台运行
  • –name dvwa：容器命名
  • -p 8080:80：映射 Web 服务端口
  • -p 3306:3306：映射 MySQL 端口，便于外部连接
  • –restart always：随系统自动重启

⚠️ 注意：docker run 的参数必须写在同一条命令中，不可拆开写。

确认docker是否开机自动

docker inspect --format='{{.HostConfig.RestartPolicy}}' dvwa

如果想要改成开机自动启动

docker update --restart=always dvwa

如果想要关闭开机自启动

docker update --restart=no dvwa

📦 Docker 常用命令速查表

类别 命令 用途说明
容器生命周期 docker run 创建并启动容器，常用：-d -p --name -v
docker start/stop/restart 后面加具体的容器名字 控制容器运行状态
docker exec -it <name> /bin/bash 进入容器内部
docker rm 删除容器（-f 强制）
日志与状态 docker ps -a 查看容器列表
docker logs -f 实时查看日志
docker stats 查看资源占用 /如cpu 内存占用
docker inspect 查看详细配置
镜像管理 docker pull 下载镜像
docker images 查看镜像列表
docker rmi 删除镜像
docker build 通过 Dockerfile 构建镜像
网络与存储 docker network ls 列出网络
docker volume ls 列出卷
