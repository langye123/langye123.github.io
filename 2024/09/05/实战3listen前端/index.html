构建vue3项目

  1. cmd到所属文件夹下面，直接npm create vue,注意一定是提前全局安装了vue3的vite构建工具npm install -g create-vue
  2. 注意一旦构建完成记得npm install然后我在这里遇见情况，不能完成构建，更改为淘宝源>npm config set registry https://repo.huaweicloud.com/repository/npm/
    用npm config get registry 看看当前构建源 成功之后记得 npm run dev
  3. 第三步和实战1实战2一样，安装elempentplus，和按需导入的其他两个插件

token设置

  1. 创造个轮子，来进行sessionStorage设置token 在用户登录后存储令牌，并在需要时检索和删除令牌。例如，你可能在用户登录成功后调用 setToken 来存储令牌，并在用户注销时调用 removeToken 来删除令牌。
    在需要验证用户身份时，可以使用 getToken 来检索令牌。
  2. 请求前置拦截器，和响应请求拦截 前置请求，把token放入了header里面，响应前置拦截，对状态码进行判断并且进行设置清空了sessionStorage，并强制跳转到了login
